Tech News: Build an illuminated Halloween costume

Can you believe that September is almost over and October is right around the corner!!? Since preparing Halloween costume is probably the biggest excitement during October, why not try this illuminated costume by Adafruit?

The costume is generally built with the touching sensitive Gamma Mo Microcontrollers to perform the animation that’s pre-programmed in full color. These microcontroller which runs CircuitPython are easily embedded under wearable clothes. CircuitPython is the modified Python language specially for embedded devices. What makes this topic further interesting is that Python has not been categorized as embedded language like C/C++, but if this new CircuitPython begins to form the basis of Microcontroller programing, then the “normal” in coding world might be revolutionized in the future.

Finally, if you are interested in building one yourself, there is a full instruction in the original article as well!
